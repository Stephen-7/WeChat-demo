# 小程序目录结构
|-- assets 资源
|-- image 图片资源
|-- style 样式资源

|-- subcontract 分包界页
|---- image 图片资源
|---- pages 页面
|---- webView 详情说明

|-- components 全局复用组件 注意：select为弹窗模式
|---- backTop 返回头部
|---- bluetoothprinter 蓝牙打印
|---- goodsAction 商品导航
|---- imageCropper 图片截取
|---- navCapsule 头部栏
|---- navSearch 返回搜索栏
|---- noData 暂无数据
|---- search 搜索
|---- selectCity 选择城市
|---- selectTime 选择时间
|---- swiper 轮播图
|---- tabItem tab标签页
|---- videoSwiper 视频轮播

|-- miniprogram_npm 第三方库
|-- node_modules 依赖包

|-- pages 小程序tabbar主界面
|---- home 首页 
|---- mine 个人中心
|---- login 登录页

|-- request 小程序接口和请求方法
|-- utils 小程序第三方本地包
|-- app.js 小程序逻辑
|-- app.json 小程序配置
|-- app.wxss 小程序公共样式
|-- package.json 小程序依赖包配置
|-- package-lock.json 小程序库的来源和版本号
|-- project.config.json 小程序项目配置文件
|-- sitemap.json 小程序页面索引
